# DocumentDB Samples Gallery

A collection of sample applications showcasing what you can build using [DocumentDB](https://github.com/microsoft/documentdb).

## Samples

| Sample | Description | Language |
|--------|-------------|----------|
| [book-finder-js](./book-finder-js) | AI-powered semantic book discovery using OpenAI embeddings and DocumentDB vector search | Node.js |

## What is DocumentDB?

DocumentDB is an open-source document database with native support for vector search, making it ideal for AI-powered applications that need to store and query embeddings alongside structured data.
